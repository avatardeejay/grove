# grove 🌴

A featherweight CLI version control tool. ~900kb You create a project (a list of files and folders) and then you can make saves of that project, backing up the files. You can then restore any save

## Install

**Linux / macOS:**
```sh
curl -fsSL https://yourusername.github.io/grove/install.sh | sh
```

**Windows (PowerShell):**
```powershell
irm https://yourusername.github.io/grove/install.ps1 | iex
```

Or download a binary directly from the [releases page](https://github.com/YourUsername/grove/releases).


## Screens

Menu Screen

Create - Create a new project
Grove Root - opens up your project directory
Open Project --
Exit --


Project Screen
Save - creates a backup of your files and folder.
Files - add or delist folders and files to your project.
Restore - Replace the current files/folders of project on your computer with one of your backups.
Grove - Opens up the directory for the opened project, with all its backups.
