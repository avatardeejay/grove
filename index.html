Instant Install

02

Snapshots

Save your work with a single command. Browse history, restore saves, keep moving forward.

03

Branches

Lightweight branching built for solo coders. Experiment freely. Save what works.

04

Written in Rust

Fast, safe, and tiny. No garbage collector, no runtime — just a binary that does what you ask.

05

For Solo Coders

Not built for enterprise CI pipelines. Built for one person who wants to ship without mastering a whole culture (git).

06

Part of DDE

Grove is the first element of the Disintegrated Development Environment — a tiny software suite for CLI-first developers.
